# imagecolor

You can get the top 3 image colors of the image.  

## Usage
imagecolor.pl imagefile

## Installation 
use Image::Magick;

## Include packages
WB.txt
RGB.txt  
JRGB.txt
sample1.png

## Execution example
C:\projects\imagecolor>imagecolorcalc.pl sample1.png
